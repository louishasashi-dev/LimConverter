# LimConverter
LimConverter — A versatile real-time converter with a futuristic neon display. From currencies & crypto with live rates, to everyday conversions like weight, length, area, time, speed, and data. Equipped with instant swap, popular pairs, dynamic tables, and quick search in a modern, lightweight, and responsive UI.

Play Here : [https://github.com/louishasashi-dev/LimConverter](https://louishasashi-dev.github.io/LimConverter/)
